个人信息格式为：属性名，属性值。

关于记忆记录的说明：
记忆格式为：序号, 类型, 记忆内容。
序号顺序表明记忆产生的顺序，序号越小代表记忆产生的越早。
记忆类型可分为：观察、决策、具体行动。
[观察]：是你观察到的客观现实环境。
[决策]：是你计划做但还未实际做的行动。
[相关属性集合]：是你对个人信息所有属性的“决策相关性评估”，评估得分越高，表明你是由于该属性才做出的相应决策。
[决策实施计划]：是你为了完成决策制定的实施计划。
[相关行动集合]：是你对具体行动部分的“决策相关性评估”，评估得分越高，表明你在实施计划时越会选择该具体行动。
[具体行动]：是你从相关行动集合中选出的用于执行你决策实施计划当前步骤的具体行动。
[具体行动结果]: 是你执行最近一次的具体行动返回的结果。
[具体行动结果评估]：你执行一次具体行动之后，会进行一次观察，你根据观察结果评估具体行动，评估结果为：具体行动的结果取得了预想的效果、具体行动的结果未取得预想的效果。
[决策实施状态]：是你对最近一条决策实施计划状态的评估，评估分为三类[正常完成]、[未完成]、[终止该计划]
[高级思考]：当你的记忆记录每周期性的增加到一定数量时，你会进行一次高级思考，从而对最近的过往一段时间内的记忆进行思考总结。高级思考总结出了这段时间内最重要、最高层次的记忆内容。

关于指导意见的说明：
指导意见具有重要的参考价值，你需要在之后的活动中重视指导意见。
指导意见的格式为(指导意见下发的时间，指导意见内容)，时间序号越小，代表指导意见下发的时间越早。
指导意见之间用%进行分割。
